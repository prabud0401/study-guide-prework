HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML5 is the latest version of HTML and comes with new semantic elements like
  • The DOCTYPE html declaration at the beginning of the document defines the HTML version being used.
  • Attributes provide additional information about HTML elements. For example, the element has a src attribute for the image source.

CSS

  • CSS stands for Cascading Style Sheets.
  • It's used to style and layout web pages, enhancing the visual presentation.
  • Selectors in CSS help target specific HTML elements for styling.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control system that tracks changes in code over time.
  • Branching in Git allows for parallel development and testing of features.
  • Commands like git add, git commit, and git push are essential in Git workflows.

JavaSript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is a scripting language that adds interactivity to web pages.
  • Variables in JavaScript store data, and functions perform actions.
  • The DOM (Document Object Model) is a crucial concept for manipulating web page elements.
